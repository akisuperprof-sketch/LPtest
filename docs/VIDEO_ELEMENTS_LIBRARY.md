# 動画演出・表現要素 完全ライブラリ

映像制作（LP動画、広告、SNS、Youtubeなど）において、感情や意図を正確に伝えるための「演出の引き出し」を体系化したライブラリです。
企画時の構成案作成、AIへのプロンプト指示、編集者へのディレクション、Remotion実装時のパラメータ選定などに活用してください。

---

## 🚀 シーン・目的別 クイックリファレンス

演出に迷ったときは、ターゲットの感情や媒体に合わせて以下から選択してください。

| 目的・シーン | 推奨エフェクト・演出 |
| :--- | :--- |
| **高級感・ラグジュアリー**<br>(不動産、ホテル、宝飾品) | **VFX**: レンズフレア、グロー、スローモーション<br>**Transition**: クロスフェード、フレアワイプ<br>**Text**: フェードイン、広めの字間（Tracking） |
| **信頼・誠実**<br>(教育、医療、BtoB) | **VFX**: 固定カメラ、ホワイトバランス調整<br>**Transition**: ハードカット、ホワイトフェード<br>**Text**: アンダーライン描画、フェードイン |
| **SNS・Z世代・トレンド**<br>(求人、カフェ、アプリ) | **VFX**: ハンドヘルド、ネオン、RGBスプリット<br>**Transition**: ズームイン/アウト、スライド、スリット<br>**Text**: ポップアップ、バウンド、キネティックタイポ |
| **不安・課題提起 (Problem)**<br>(LP冒頭、悩み訴求) | **VFX**: グレイン(ノイズ)、彩度低下、ドリーズーム<br>**Transition**: ブラックアウト、グリッチ<br>**Text**: タイピング、ノイズ出現 |
| **解決・高揚感 (Solution)**<br>(LP後半、商品登場) | **VFX**: パーティクル(キラキラ)、ライトリーク、サチュレーションUP<br>**Transition**: フラッシュカット、ズームイン<br>**Text**: 3D化、スタンプ押し出し |

| | **フリーズフレーム** | 一時停止＋拡大 | 印象付け、キャラクター紹介、ナレーション解説 |

## 2. トランジション - 場面転換の文法

| カテゴリ | 名称 | 特徴 | 適した用途 |
| :--- | :--- | :--- | :--- |
| **基本** | **ハードカット** | エフェクトなしで瞬時に切替 | 会話、テンポ重視、YouTuber動画 |
| | **ジャンプカット** | 同一画角で時間だけ飛ばす | スピーディーな情報伝達、無言カット |
| **フェード** | **クロスフェード** | 透明度で重なりながら切替 | 時間経過、回想、情緒的なシーン |
| | **ホワイトフェード** | 白を経て切り替わる | 夢落ち、明るい場面転換、フラッシュバック |
| | **ブラックフェード** | 黒を経て切り替わる | 章の区切り、終了、暗転 |
| **モーション** | **スライド (Whip)** | 高速で横/縦にスライド | 方向性の提示、スピード感、比較 |
| | **ズームイン/アウト** | カメラが飛び込む/引く動き | 没入感維持、シーンの関連付け（マクロ→全体） |
| **特殊** | **グリッチ** | デジタルノイズと共に切替 | テック系、ホラー、急展開 |
| | **マスクトランジション** | 柱や人を境界線にして切替 | 魔法のようなシームレスな演出、Vlog |
| | **フリーズ→カット** | 直前の絵が止まって次へ | MV、スタイリッシュな紹介 |

## 4. 編集・演出テクニック - 組み合わせの魔法

1.  **マッチカット (Match Cut)**
    *   **手法**: 前のカットと次のカットで「形」「動き」「色」が似ているものを繋ぐ。（例: 丸い時計 → 丸い月）
    *   **効果**: 芸術的な連続性、時間の経過を美しく表現。
2.  **Jカット / Lカット**
    *   **手法**: 音声と映像の切り変わりタイミングをずらす。（Jカット＝次のシーンの音が先に聞こえる）
    *   **効果**: 映像のつなぎ目を意識させない、プロフェッショナルなフロー。
3.  **リズム編集 (Beat Sync)**
    *   **手法**: BGMのバスドラムやスネアに合わせてカットを割る。
    *   **効果**: 視聴者が心地よく見続けられる、中毒性を生む。
4.  **コントラスト強調**
    *   **手法**: 「静」のシーンの直後に「動」のシーンを配置する（またはその逆）。
    *   **効果**: ギャップによる衝撃、飽きさせない構成。
5.  **視線誘導**
    *   **手法**: 画面内の動きや指差し、視線の向きに合わせて次のカットの配置を決める。
    *   **効果**: ストレスのない視聴体験。
