# 動画作成・品質向上ワークフロー (目標: TV CM / SNS広告レベル)

現状の「静止画スライドショー（70点）」から、**「プロ仕様のTV CM / SNS広告レベル（90点以上）」** へ引き上げるための標準化プロセスです。
**BGMとSE（効果音）の実装は「オプション」ではなく「必須要件」とします。**

## 1. 共通: 高品質化のためのコア技術要件 (Core Requirements)
Remotion実装時に以下の要素を必ず組み込むこと。

### 🔊 Audio Design (必須・最重要)
**映像のクオリティの50%は「音」で決まる。無音や単調なBGMは禁止。**
- **BGM (Background Music)**:
  - 動画のターゲット感情（安心、ワクワク、緊急性）に完全に合致したものを選定。
  - **Beat Sync**: 映像のカット切り替えや文字の出現を、BGMのリズム（ビート）に必ず合わせる。
- **SE (Sound Effects)**:
  - **動きには必ず音をつける**:
    - テキスト出現時: 「シュッ (Whoosh)」「ポン (Pop)」「カチッ (Click)」
    - 場面転換時: 「ブゥン (Transition Whoosh)」「キラキラ (Sparkle)」
    - 注目ポイント: 「ドン (Impact)」「ピンポン (Chime)」
  - TV CMやYouTuber動画のような「心地よいリズム」を音で作る。

### 🎥 Visual Motion
- **Ken Burns Effect (パン・ズーム)**:
  - 静止画をただ置くのではなく、`scale` と `translate` を `interpolate` で動かし、映像に「動き」を与える。
  - 指示: 「画像はゆっくりと1.0倍から1.1倍へズームインさせてください」
- **Kinetic Typography (動く文字)**:
  - テキストはフェードインだけでなく、一文字ずつの出現や、スライドイン、バウンスなどを使用する。
  - ライブラリ: `@remotion/text` や `framer-motion` のロジックを参考にする。
- **トランジション (画面切り替え)**:
  - 単純なカットではなく、`Slide`, `Fade`, `Wipe` などを意図を持って使い分ける。

## 2. 不動産動画 (Real Estate) 制作フロー

### ターゲット品質
「内見に行きたくなる」「住んでいるイメージが湧く」没入感のある動画。

### 手順
1.  **素材準備 (Assets)**
    - 写真、間取り図、スペック推奨。
    - **Audio**: ラウンジ系/Jazz HipHop等の「高級感のあるBGM」。ドアが開く音、足音などの環境音SEも可。
2.  **構成案 (Storyboard)**
    - **Intro (0-3s)**: 物件外観 + ロゴ。高級感のあるSE（「シュアァ...」）と共にフェードイン。
    - **Highlight (3-15s)**: Ken Burns EffectでLDKを見せる。情報を出すときは「ポン」とSEを入れる。
    - **Tour (15-40s)**: 玄関 -> 水回り -> 寝室。切り替えはBGMの小節の区切りで行う。
    - **Outro (40-45s)**: 会社ロゴと共に「カチッ」と締めるSE。
3.  **Remotion実装指示 (Prompt Example)**
    > 「不動産物件の画像をスライドショーにします。BGMは高級感のあるラウンジ系。画像切り替えはBGMのビートに合わせてください。テキストが表示されるたびに、上品な『Pop』音(SE)を鳴らしてください。」

## 3. ニッチLP動画 (Marketing LPs) 制作フロー

### ターゲット品質
「悩みに共感し、解決策に希望を持ち、クリックしたくなる」感情訴求型の動画（TV通販/ウェブ広告風）。

### 手順
1.  **素材準備 (Assets)**
    - Unsplash画像、キャッチコピー。
    - **Audio**:
      - 前半（悩み）: 不安を煽る重低音BGM or 無音 + ノイズ音。
      - 後半（解決）: 開放感のある明るいBGM。キラキラ音、ファンファーレSE。
2.  **構成案 (Storyboard) - PASONA**
    - **Problem (0-5s)**: 「悩み」テキスト出現時に「ドン！」というImpact SE。
    - **Solution (10-20s)**: 商品登場時に「ジャジャーン（Reveal）」SEと、明るいBGMへの転換。
    - **Benefit (20-35s)**: メリット提示ごとに「ピンポン（正解音）」や「キラキラ」SEで肯定感を演出。
    - **Action (35-45s)**: ボタンがクリックされるアニメーション + 「カチッ」SE。
3.  **Remotion実装指示 (Prompt Example)**
    > 「前半は不安なBGMで『ドン』というSEと共に悩みを強調。後半は商品登場と共に明るい曲に切り替え、『キラキラ』したSEを多用して解決策を演出してください。」

## 4. 改善サイクル (PDCA)
- 生成された動画（mp4）を確認し、以下の観点で採点する。
  - **Audio Quality (最重要)**: BGMとSEが入っているか？ 音量は適切か？ タイミングは合っているか？
  - **リズム感**: 間延びしていないか？
  - **可読性**: 文字が背景に溶け込んでいないか？
  - **感情**: ターゲットが見て「自分ごと」と思えるか？
- 修正指示を出し、パラメータ（duration, fps, colors, audio volume）を微調整する。
